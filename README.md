# IP Address Tracker

This is a solution to the [IP Address Tracker challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/ip-address-tracker-I8-0yYAH0). The application allows users to search for any IP addresses or domains and see the key information and location on an interactive map.
